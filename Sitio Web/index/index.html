Esto principalmente es un proyecto de la preparatoria, de la carrera de programacion el cual es una pagina web de ventas en este caso cree "GAMETILT".

El cual es una pagina un poco precaria para el funcionamiento correcto de una pagina web profesional pero es un proyecto pequeño el cual costo un poco de hacer.

Tuve que hacerlo en la IDE de Brackets usando los idiomas de HTML, CSS y Js los cuiales pueden ser de mas utilidad para la facilidad de crear la pagina.
